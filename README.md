# ChartsDemo

Implementing charts (Bar/Column) using PrimeNG and Ngx Charts with pagination in Angular.

![PrimeNG Chart](/img/primeng.PNG)
![Ngx Chart](/img/ngx-2.PNG)

## Features:
1. Bar/Column Chart
2. Column Click event handler
3. Pagination (for large data sets)

## Steps:
1. CMD `npm install`
2. CMD `npm start`
